Vanliga frågor

Hur lägger jag till ett mätvärde?

Du kan lägga till ett nytt mätvärde antingen 1. genom att trycka på dagen eller 2. genom att trycka på + uppe till höger. Båda alternativen visas med pilar i bilden nedan.

Kan jag registrera flera gånger per dag?

Ja. Du kan lägga till många mätvärden per dag, som visas ovan. Tryck bara på dagen igen, eller på +, för att göra detta.

Hur tar jag bort ett mätvärde?

För att ta bort ett mätvärde kan du svepa åt vänster på mätvärdet och trycka på Ta bort, eller trycka på mätvärdet för att öppna det och sedan trycka på papperskorgsikonen uppe till vänster på skärmen. Båda metoderna visas nedan:

Varför ser jag ett sentimentvärde ovanför mina anteckningar? Är detta privat?

När dina anteckningar innehåller 8 eller fler ord visas en emoji ovanför dem för att indikera det beräknade sentimentet. Den här funktionen kan hjälpa dig förstå om ditt skrivande uppfattas som alltför negativt eller positivt, eller påminna dig om vad som fungerar eller inte, och fungerar som en guide för ditt skrivande. Sentimentet beräknas helt och hållet på din enhet med hjälp av Apples Natural Language API, så inga data lämnar din enhet, vilket säkerställer fullständig integritet. Om du föredrar det kan du inaktivera den här funktionen i Inställningar under Allmänt.

Hur skriver jag ut?

När du trycker på Exportera (som visas nedan) visas ett delningsark. Bläddra nedåt på det och tryck på Skriv ut (som visas nedan). Du behöver en trådlös skrivare som är kompatibel med Apple AirPrint för att skriva ut trådlöst från din iPhone eller iPad.

Om du inte har en trådlös AirPrint-kompatibel skrivare kan du köpa en här.

Kan jag dela medicinsk information som jag spårar med min läkare?

Ja. Appen gör det enkelt att göra detta. Tryck på delningsknappen (som visas nedan), välj PDF och välj sedan din e-postapp. PDF-rapporten bifogas till ett e-postmeddelande som du sedan kan skicka till din läkare.

Hur kan jag dölja medelvärden eller summor i Dagbok?

Du kan dölja medelvärden och summor i visningsalternativen, som visas nedan. Det visar då bara dina inmatade värden och inte medelvärden. Det kan vara användbart om du tycker att det är förvirrande att ha medelvärden/summor bland dina inmatade data.

Kan jag exportera till Excel eller liknande kalkylprogram?

Ja. Följ instruktionerna ovan, men välj Excel eller CSV istället för PDF som utdataformat. Spara sedan filen på din enhet eller skicka den via e-post till dig själv.

Excel-formatet är mer tillförlitligt, medan CSV kan vara mer allmänt kompatibelt – båda bör öppnas i Excel och liknande program, som Apple Numbers eller Google Sheets.

Till vem kan jag skicka mina data?

Du kan skicka dem via e-post till vem du vill. Du kanske vill regelbundet skicka dina data till dig själv för säkerhets- och säkerhetskopieringssyften. Du kan också spara PDF- eller CSV-filen på din enhet för senare användning.

Hur scrollar jag direkt till aktuell dag i månadsvy?

Dubbeltryck på Dagboksikonen i flikfältet längst ned på skärmen.

Hur visar jag data från tidigare månader?

För att visa data från en tidigare månad (till exempel septemberdata när du befinner dig i oktober):

  • Gå till fliken Dagbok
  • Tryck på bakåtknappen uppe till vänster – detta tar dig till månadsval
  • Välj den månad du vill visa (t.ex. september)
  • Om du inte ser dina data, tryck på dagboksalternativknappen (den 2:a knappen längs toppen, bredvid bakåtknappen) och se till att alla visningsalternativ är inställda på "Visa"

Dina data sparas alltid och är tillgängliga – du kan navigera mellan månader när som helst för att se dina historiska mätvärden.

Om du fortfarande inte ser dina data efter att du kontrollerat månadsvisningen: Gå till Inställningar > Data > Hantera säkerhetskopior och leta efter eventuella säkerhetskopior från den månad du försöker visa (eller tidigt i följande månad). Om du hittar en säkerhetskopia från den tidsperioden kan du återställa från den för att återfå dina data. Det kan hjälpa om du har installerat om appen eller om det uppstod ett synkroniseringsproblem.

Hur ändrar jag tid eller datum på ett mätvärde?

Öppna ett mätvärde och tryck på datumet eller tiden längst upp på skärmen, som visas nedan. En kalender visas där du kan välja rätt datum och sedan trycka på Spara.

Tips: Om du alltid vill skapa ett mätvärde för dagens datum, tryck på + bredvid aktuell dag, eller på + uppe till höger på skärmen. Om ett mätvärde hamnade på fel datum kan det bero på att + trycktes bredvid en annan dag – vissa användare använder detta för historiska mätvärden eller framtida planering.

Hur säkra är mina data?

Dina data krypteras när de lagras på din enhet. Det innebär att även om din enhet stjäls kan ingen komma åt dem.

Var lagras mina data?

På din enhet och i iCloud. Den här appen använder samma teknik som Apple Notes och Foton för att spara och synkronisera dina data mellan enheter, vilket innebär att det är mycket tillförlitligt och säkert.

Kan någon annan se mina privata data?

Nej. Data finns enbart på din enhet och i iCloud, som körs på servrar hanterade av Apple.

Varför behöver mina data sparas i iCloud?

iCloud gör det möjligt att säkerhetskopiera dina data och synkronisera dem mellan dina enheter. Om du förlorar din telefon eller skaffar en ny kan dina data återställas från iCloud.

Varför synkroniseras inte mina data mellan mina enheter?

Dina data bör synkroniseras automatiskt mellan dina enheter. Se till att:

  • Du är inloggad på samma iCloud-konto på alla enheter.
  • Alla iOS-enheter har samma version av iOS (helst den senaste).
  • Se till att iCloud Drive är På på alla enheter (i Inställningar > Ditt namn > iCloud > iCloud Drive, som visas nedan.

Behöver jag ha ett abonnemang för varje enhet?

Ett abonnemang tar bort annonser på alla enheter. Om du ser annonser på ytterligare enheter, gå till Inställningar > Ta bort annonser och tryck på Återställ köp. Se till att varje enhet är inloggad på samma iCloud-konto.

Hur fungerar säkerhetskopieringar?

Säkerhetskopieringar skapas automatiskt varje vecka som standard (även om du kan stänga av detta i Inställningar). Du kan också skapa manuella säkerhetskopieringar i Inställningar > Data.

Hur många säkerhetskopieringar skapas? Bör jag ta bort dem?

Det finns ingen gräns för antalet säkerhetskopieringar som skapas. Säkerhetskopieringar kan gå år tillbaka och tar generellt sett mycket lite utrymme, så att ta bort dem gör inte stor skillnad för enhetens lagringsutrymme. Om du behöver frigöra utrymme är det bättre att ta bort tyngre filer på din enhet, till exempel foton och videor.

Synkroniseras säkerhetskopieringar till iCloud?

För närvarande synkroniseras inte säkerhetskopior via iCloud, men din aktuella data gör det. Det betyder att om du skaffar en ny iPhone eller iPad visas din aktuella data där också, men inte dina säkerhetskopior, eftersom varje enhet behåller sina egna lokala säkerhetskopior.

Detta innebär att om du till exempel tar bort en post på din iPhone och den försvinner från iCloud och din telefon, kan du fortfarande gå tillbaka till din iPad och återställa från en av dess äldre säkerhetskopieringsögonblicksbilder.

Behöver jag ett konto för att använda den här appen?

Nej. Du kan börja använda den direkt utan att behöva logga in eller registrera dig. Appen använder ditt iCloud-konto för att automatiskt synkronisera dina data till andra enheter via Apples infrastruktur. Det innebär att det är snabbt, tillförlitligt och säkert.

Kan jag använda den här appen på både min iPhone och iPad?

Ja, och dina data synkroniseras automatiskt mellan dem om du är inloggad på samma iCloud-konto. I iPad-versionen har du mer utrymme för anteckningar och kan se fler poster.

Finns det en Mac-version av appen?

Ja. Appen finns tillgänglig för Apple Silicon-Macar (M1 och senare). I Mac-appen kan du ändra fönstrets storlek för att se mer data på större skärmar. Du kan ladda ned Mac-appen här på din M1 eller senare Mac.

Vad är skillnaden mellan Dagbok, Trender och Statistik?

Dagbok är den huvudsakliga dagboks-/loggsektionen som låter dig mata in data och visa den månadsvis/årsvis/under hela livstiden, med genomsnitt och totaler.

Trender låter dig identifiera mönster i de data du registrerar i Dagboken så att du kan agera på dem.

Statistik låter dig se dina data i kronologisk ordning (nyaste posten först) utan månads- eller årsgruppering, och låter dig se genomsnitt och totaler mellan valfria två datum.

Bör jag logga ut eller stänga ned för att spara telefonens minne och batteri?

Det finns absolut ingen anledning att logga ut – enheten laddar automatiskt ur appar/resurser från minnet vid behov.

Moderna iPhones och iPads har mycket energieffektivt minne, så appar kan stanna kvar i minnet länge utan några negativa effekter. Åtkomsten till detta minne är säker och skyddas redan av Face ID, Touch ID eller en lösenkod.

Om du ändå vill stänga en app (till exempel om den har slutat svara) kan du följa Apples instruktioner här.

Det finns heller ingen anledning att logga in, eftersom appen använder ditt iCloud-konto och redan kan synkronisera/säkerhetskopiera data mellan dina enheter, ungefär som Foton- och Anteckningar-apparna gör, utan att behöva dela några identifierande personuppgifter.

Ja. Så snart du trycker på sidoknappen på enheten för att låsa den som du normalt gör, kan ingen komma åt din app eller dina data medan den är låst.

Blodtrycksappen

Ansluter appen till min blodtrycksmonitor via Bluetooth?

Nej, appen ansluter inte direkt till blodtrycksmonitorer via Bluetooth. Istället har vi fokuserat på att göra manuell inmatning enkel genom AI-kameraigenkänning. Rikta helt enkelt din iPhones kamera mot monitorns display efter att ha tagit en mätning, och appen läser av siffrorna automatiskt. Detta fungerar med i princip vilken blodtrycksmonitor som helst med en digital display, oavsett märke eller modell. Ingen parkoppling krävs, ingen specifik enhetskompatibilitet att bekymra sig om, och det tar bara några sekunder. Den här funktionen är alltid gratis och obegränsad.

Hur fungerar AI-kameraskanning?

Efter att ha tagit ditt blodtryck öppnar du appen och trycker på "+"-knappen, välj sedan "Skanna med kamera." Håll telefonen still cirka 15–30 cm från monitorns display och se till att siffrorna är tydligt synliga och välbelysta. Appen läser automatiskt av systoliska, diastoliska och pulsvärden. För bästa resultat, se till att belysningen är god och håll telefonen stilla under skanningen. Detta fungerar med de flesta digitala monitorer, inklusive OMRON-, Microlife- och Beurer-märken.

Varför ser jag dubblettinlägg?

Detta orsakas vanligtvis av Apple Hälsas synkroniseringsinställningar. Det vanligaste problemet är att enbart skrivbehörighet är aktiverad för Apple Hälsa, men inte läsbehörighet. För att åtgärda detta: Gå till Inställningar > Data > Apple Hälsa och se till att BÅDE läs- OCH skrivbehörigheter är aktiverade för blodtryck och hjärtfrekvens – inte bara skrivbehörighet. Appen behöver läsbehörighet för att kontrollera om en mätning redan finns innan den importeras från Hälsa.

Kan jag spåra mätningar för flera familjemedlemmar?

En taggfunktion lanseras inom de närmaste månaderna som gör att du kan spåra mätningar för flera personer på en enhet. Du kommer att kunna tagga varje post med en persons namn och sedan generera individuella rapporter för varje familjemedlem.

Mina data synkroniseras inte mellan min iPhone och iPad. Varför?

Kontrollera först att du är inloggad på samma iCloud-konto på båda enheterna. Gå till Inställningar > [Ditt namn] och kontrollera Apple-ID:t. Kontrollera sedan att iCloud Drive är aktiverat på båda enheterna (Inställningar > [Ditt namn] > iCloud > iCloud Drive). Dina data synkroniseras automatiskt via iCloud, men det krävs en internetanslutning och det kan ta några minuter. Prova att öppna appen på varje enhet och dra ned för att uppdatera.

Vad ingår i prenumerationen? Vad är gratis?

Alla grundläggande spårningsfunktioner är helt gratis: obegränsade poster, AI-kameraskanning (alltid gratis), alla diagram och trender, synkronisering med Apple Hälsa samt PDF/CSV-exporter. Prenumerationen ger: annonsfi upplevelse, förbättrad AI-chatt (100 användningar per dag jämfört med 10 gratis) och tillgång till avancerade AI-funktioner. Alla framtida premiumfunktioner ingår för prenumeranter.

Vad är MAP och bör jag spåra det?

MAP (medelartärtryck) representerar det genomsnittliga trycket i dina artärer under en hjärtcykel. Det beräknas automatiskt för varje mätning. MAP är särskilt viktigt för LVAD-patienter (Left Ventricular Assist Device) och andra med specifika hjärt-kärlsjukdomar. För att aktivera MAP-visning, gå till Inställningar > Blodtryck > Visa medelartärtryck (MAP). Anpassningsbara MAP-intervall lanseras i version 2.8.x.

Hur använder jag anteckningsmallar för att spara tid?

När du skriver en anteckning (till exempel "Morgonmätning – innan medicin") trycker du på knappen "Mall" längst ned till vänster i anteckningsfältet och sparar den som en mall. Nästa gång du registrerar ett värde trycker du på "Mall" igen och väljer bland dina sparade mallar – texten infogas direkt. Det här är perfekt för återkommande anteckningar som "Morgon – innan medicin" eller "Kväll – efter middag."

Vad är för högt eller lågt blodtryck?

Ditt blodtryck bör normalt ligga mellan 90–140 för det systoliska värdet och 60–90 för det diastoliska värdet. Om det ligger utanför detta intervall bör du tala med en läkare.

Vad är ett idealiskt blodtryck?

Enligt NHS är ett idealiskt blodtryck mellan 90/60 mmHg och 120/80 mmHg. Det kan dock variera beroende på kön, ålder och andra faktorer. Det är därför bäst att rådfråga din läkare för en korrekt bedömning av ditt blodtryck utifrån dina individuella omständigheter.

Hur mäter jag mitt blodtryck?

Blodtrycket mäts med en digital blodtrycksmonitor. Här hittar du blodtrycksmätare som fungerar utmärkt med Kamera-skanning AI-funktionen. Appen kan automatiskt läsa av värden från de flesta digitala mätare, inklusive märkena OMRON, Microlife och Beurer.

Vad är Trender till för?

Med Trender tar appen dina data och anteckningar och visar vad som påverkar dig och hur. Till exempel kan du se att du mediterade 20 gånger förra året och att meditation sänker ditt blodtryck. Omvänt kan du se att varje gång du träffar Bob bråkar ni, och han höjer ditt blodtryck. Ju mer du använder anteckningar, desto mer användbar kan Trender bli.

Kan du förklara Trender med ett exempel?

Du springer till exempel 10 gånger. 5 av gångerna sjunker ditt blodtryck med 10, och de övriga gångerna sjunker det med 5. Trender visar att löpning i genomsnitt hjälper till att sänka ditt blodtryck med ungefär 7,5 enheter.

Feeltracker Humördagbok-app

Vad är Trender till för?

Med Trender tar appen dina data och anteckningar och visar vad som påverkar dig och hur. Till exempel kan du se att du träffade Jane 20 gånger förra året och att Jane i genomsnitt gör dig väldigt glad. Omvänt kan du se att varje gång du träffar Bob bråkar ni, och han gör dig olycklig. Ju mer du använder anteckningar, desto mer användbar kan Trender bli.

Kan du förklara Trender med ett exempel?

Du träffar till exempel Jane 10 gånger. 5 av gångerna gjorde hon dig väldigt glad (+2), och 5 av gångerna gjorde hon dig ganska glad (+1). Trender visar att Jane i genomsnitt gör dig ganska (+1,5) glad.

Daglig dagbok och anteckningsapp

Vad är Trender till för?

Med Trender tar appen dina data och anteckningar och visar vilka ord du har mest i tankarna. Till exempel kan du se att du använde ett ord 20 gånger förra året. Omvänt kan du se att du använde ett annat ord 50 gånger – betydligt oftare. Ju mer du använder anteckningar, desto mer användbar kan Trender bli.

Kan du förklara Trender med ett exempel?

Du skriver till exempel om Jane 120 gånger i dina anteckningar, mer än om ditt jobb som du skriver om 50 gånger. Trender visar i det här fallet tydligt att det som låg mest i dina tankar var Jane.

Sömndagboksappen

Vad är Trender till för?

Med Trender tar appen dina data och anteckningar och visar vad som påverkar dig och hur. Till exempel kan du se att du drack mycket kaffe förra veckan och att det fick dig att sova sämre. Omvänt kan du se att varje gång du äter sallad eller tränar på gymmet sover du bättre. Ju mer du använder anteckningar, desto mer användbar kan Trender bli.

Kan du förklara Trender med ett exempel?

Du dricker till exempel kaffe 10 gånger. 5 av de gångerna sov du 2 timmar för lite, och de övriga 5 gångerna sov du 1 timme för lite. Trender visar att kaffe i genomsnitt gör att du sover 1,5 timmar för lite.

Vikt- och kroppsfett matdagbok-app

Vad är Trender till för?

Med Trender tar appen dina data och anteckningar och visar vad som påverkar dig och hur. Till exempel kan du se att du åt snabbmat 20 gånger förra året och att du med tiden ökar i vikt när du äter snabbmat. Omvänt kan du se att varje gång du äter sallad eller tränar på gymmet minskar du i vikt. Ju mer du använder anteckningar, desto mer användbar kan Trender bli.

Kan du förklara Trender med ett exempel?

Du äter till exempel munkar 10 gånger. 5 av gångerna ökar din vikt med 0,1 kg. De övriga 5 gångerna ökar din vikt med 0,3 kg. Trender visar att munkar i genomsnitt gör att din vikt ökar med 0,2 kg.

Hur mäter jag kroppsfett?

Kroppsfett mäts av speciellt utformade kroppsvågar som mäter kroppssammansättning utöver vikt. Här hittar du ett exempel på en sådan våg från Omron. Det finns även andra alternativ från andra märken – det du ska leta efter är "Body Composition" (kroppssammansättning) på en vanlig våg.

Observera också att även om du bara registrerar vikt, kommer du fortfarande att se ett BMI-värde (till vänster om vikten, färgkodat) som ger en övergripande indikation på om du har en hälsosam vikt (grönt är bra, rött innebär övervikt).
