Användarvillkor

Genom att ladda ner eller använda appen gäller dessa villkor automatiskt för dig – du bör därför se till att du läser dem noggrant innan du använder appen. Du får inte kopiera eller modifiera appen, någon del av appen eller våra varumärken på något sätt. Du får inte försöka extrahera appens källkod, och du bör inte heller försöka översätta appen till andra språk eller göra derivatversioner. Appen i sig, och alla varumärken, upphovsrätter, databasrättigheter och andra immateriella rättigheter relaterade till den, tillhör fortfarande Custom Arts Ltd.

Custom Arts Ltd är engagerat i att se till att appen är så användbar och effektiv som möjligt. Av den anledningen förbehåller vi oss rätten att göra ändringar i appen eller att ta betalt för dess tjänster när som helst och av vilken anledning som helst. Vi kommer aldrig att debitera dig för appen eller dess tjänster utan att göra det helt klart för dig exakt vad du betalar för.

Feeltracker-appen lagrar och bearbetar data med hjälp av Apples iCloud-tjänst. All data som genereras av appen synkroniseras mellan dina enheter via iCloud, och Custom Arts Ltd har inte tillgång till dessa data. Du kan styra iCloud-datalagring via enhetens iCloud-inställningar. Det är ditt ansvar att hålla din telefon och åtkomst till appen säker. Vi rekommenderar att du inte jailbreakar eller rootar din telefon, vilket innebär att ta bort programvarubegränsningar som införts av enhetens officiella operativsystem. Det kan göra din telefon sårbar för skadlig programvara/virus/skadliga program, äventyra telefonens säkerhetsfunktioner och det kan innebära att Feeltracker-appen inte fungerar korrekt eller inte alls.

Appen använder tredjepartstjänster som deklarerar sina egna användarvillkor.

Länkar till användarvillkoren för tredjepartstjänster som används av appen:

  • AdMob
  • Google Analytics for Firebase
  • Firebase Crashlytics
  • RevenueCat

Du bör vara medveten om att det finns vissa saker som Custom Arts Ltd inte tar ansvar för. Vissa funktioner i appen kräver att appen har en aktiv internetanslutning. Anslutningen kan vara Wi-Fi eller tillhandahållas av din mobiloperatör, men Custom Arts Ltd kan inte ta ansvar för att appen inte fungerar med full funktionalitet om du inte har tillgång till Wi-Fi eller om du inte har någon datakvot kvar.

Om du använder appen utanför ett område med Wi-Fi bör du komma ihåg att villkoren i avtalet med din mobiloperatör fortfarande gäller. Som ett resultat kan du debiteras av din mobiloperatör för kostnaden för data under tiden du är ansluten medan du använder appen, eller andra tredjepartsavgifter. Genom att använda appen tar du ansvar för alla sådana avgifter, inklusive dataroamingavgifter om du använder appen utanför ditt hemterritorium (dvs. region eller land) utan att stänga av dataroaming. Om du inte är den som betalar räkningen för enheten som du använder appen på, var medveten om att vi antar att du har fått tillstånd från den som betalar räkningen för att använda appen.

På samma sätt kan Custom Arts Ltd inte alltid ta ansvar för hur du använder appen. Till exempel måste du se till att din enhet förblir laddad – om batteriet tar slut och du inte kan slå på den för att använda Tjänsten kan Custom Arts Ltd inte acceptera ansvar.

Med avseende på Custom Arts Ltd:s ansvar för din användning av appen är det viktigt att tänka på att även om vi strävar efter att se till att den är uppdaterad och korrekt vid alla tillfällen, förlitar vi oss på tredje parter för att tillhandahålla information till oss så att vi kan göra den tillgänglig för dig. Custom Arts Ltd accepterar inget ansvar för någon förlust, direkt eller indirekt, som du upplever som ett resultat av att helt förlita dig på appens funktionalitet.

Vid någon tidpunkt kanske vi vill uppdatera appen. Appen är för närvarande tillgänglig på iOS – kraven för systemet (och för eventuella ytterligare system som vi beslutar att utöka tillgängligheten av appen till) kan ändras, och du måste ladda ner uppdateringarna om du vill fortsätta använda appen. Custom Arts Ltd lovar inte att appen alltid kommer att uppdateras så att den är relevant för dig och/eller fungerar med den iOS-version som du har installerad på din enhet. Du lovar dock att alltid acceptera uppdateringar av applikationen när de erbjuds dig. Vi kan också vilja sluta tillhandahålla appen och kan avsluta användningen av den när som helst utan att ge dig uppsägningstid. Om vi inte säger annat kommer vid eventuell uppsägning (a) de rättigheter och licenser som beviljats dig i dessa villkor att upphöra; (b) du måste sluta använda appen och (om det behövs) radera den från din enhet.

OpenAI-tjänster och ansvar

Vår app använder OpenAI:s API för att tillhandahålla AI-drivna funktioner, såsom att generera återkoppling eller innehåll baserat på användarinmatning. Observera att även om vi strävar efter noggrannhet och användbarhet kan innehåll som genereras av OpenAI vara ofullständigt eller felaktigt. Därför bör alla beslut som fattas baserat på AI-genererat innehåll fattas med försiktighet, och vi accepterar inte ansvar för eventuella resultat som uppstår från användningen av OpenAI:s tjänster.

Genom att använda AI-funktionerna i vår app godkänner du följande:

  • Du förstår att AI-genererat innehåll endast är i informationssyfte och inte ersätter professionell rådgivning (t.ex. medicinsk, juridisk eller ekonomisk rådgivning).
  • Du accepterar att data du skickar till AI-tjänsten kommer att bearbetas av OpenAI i enlighet med deras villkor och integritetspolicy.
  • Vi förbehåller oss rätten att uppdatera, modifiera eller inaktivera AI-funktioner i appen när som helst.

För mer information om OpenAI:s villkor, besök: OpenAI:s användarvillkor.

Ändringar av dessa användarvillkor

Vi kan uppdatera våra användarvillkor då och då. Därför rekommenderas du att granska denna sida regelbundet för eventuella ändringar. Vi meddelar dig om eventuella ändringar genom att publicera de nya användarvillkoren på denna sida.

Dessa användarvillkor gäller från och med den 19 oktober 2024.
