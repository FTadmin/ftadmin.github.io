Daily Journaling ทำงานอย่างไร

เพื่อนร่วมทางในการบันทึกส่วนตัวเพื่อจับช่วงเวลาชีวิตและค้นพบรูปแบบ

1. เขียนความคิดของคุณ

เริ่มทันที: เปิดแอปและเริ่มเขียนทันที ไม่ต้องตั้งค่า ไม่ซับซ้อน แค่ความคิดและหน้า หน้าจอสะอาดช่วยให้มุ่งเน้นการแสดงออก

สื่อหลากหลาย: เพิ่มรูปถ่ายเพื่อบันทึกความทรงจำด้วยภาพ ทุกรายการกลายเป็นภาพรวมสมบูรณ์ของวัน

เทมเพลต: ใช้เทมเพลตสำเร็จรูปสำหรับการสะท้อนรายวัน ไดอารี่ความกตัญญู การติดตามเป้าหมาย หรือสร้างเทมเพลตเองที่กำหนด

2. เห็นรูปแบบที่เกิดขึ้น

การวิเคราะห์ความรู้สึก: AI ตรวจจับโทนอารมณ์ของรายการอัตโนมัติ บวก ลบ หรือกลาง ดูแนวโน้มอารมณ์ตลอดหลายสัปดาห์และเดือน

ความถี่คำ: ค้นพบว่าเขียนเรื่องใด คน หรือธีมใดมากที่สุด แอประบุคำและวลีที่เกิดซ้ำ

ปฏิทิน: ดูประวัติการบันทึกทั้งหมดในปฏิทิน แต่ละวันแสดงตัวอย่างและตัวบ่งชี้อารมณ์

3. สะท้อนความคิดและเติบโต

AI Insights: ถามคำถามเกี่ยวกับไดอารี่: วันที่มีความสุขที่สุดเดือนที่แล้วคือวันไหน? หรือแสดงรายการที่กล่าวถึงเป้าหมาย

ติดตามการเติบโต: ทบทวนความคิด ความกังวล และแรงบันดาลใจที่เปลี่ยนแปลงไป เห็นความคืบหน้าสู่เป้าหมายที่กล่าวถึงในรายการก่อน

ค้นหาทุกอย่าง: ค้นหาคำ วลี หรือแนวคิดใดก็ได้ตลอดหลายปีของการบันทึก กรองตามวัน อารมณ์ หรือคีย์เวิร์ด

4. เก็บทุกอย่างปลอดภัย

ส่วนตัวและปลอดภัย: รายการทั้งหมดเข้ารหัสและเก็บใน iCloud ส่วนตัว มีเพียงคุณเท่านั้นที่อ่านได้

สำรองอัตโนมัติ: สำรอง JSON รายสัปดาห์อัตโนมัติเพื่อไม่ให้สูญเสียความคิด สำรองด้วยตนเองได้ตลอดเวลา

ซิงค์ข้ามอุปกรณ์: เขียนบน iPhone ต่อบน iPad ทบทวนบน Mac รายการทั้งหมดซิงค์อัตโนมัติผ่าน iCloud

คำถามที่พบบ่อย

ไดอารี่ของฉันเป็นส่วนตัวหรือไม่?

แน่นอน รายการเก็บเข้ารหัสบนอุปกรณ์และใน iCloud ส่วนตัว เราไม่เคยเห็นการเขียนของคุณ แม้แต่การวิเคราะห์ความรู้สึก AI ก็เกิดขึ้นบนอุปกรณ์ด้วย Natural Language API ของ Apple คำพูดไม่เคยออกจากโทรศัพท์

การวิเคราะห์ความรู้สึกทำงานอย่างไร?

เมื่อเขียน 8 คำขึ้นไป แอปใช้ Natural Language framework ในตัวของ Apple วิเคราะห์โทนอารมณ์ emoji ปรากฏขึ้นเพื่อแสดงว่ารายการอ่านเป็นบวก ลบ หรือกลาง ทั้งหมดนี้เกิดขึ้นบนอุปกรณ์เพื่อความเป็นส่วนตัวสมบูรณ์

การวิเคราะห์ความถี่คำคืออะไร?

ฟีเจอร์ Trends วิเคราะห์รายการทั้งหมดเพื่อแสดงคำและหัวข้อที่เขียนบ่อยที่สุด ช่วยระบุธีมที่เกิดซ้ำ ความกังวล หรือความสนใจในความคิด

เพิ่มรูปถ่ายในไดอารี่ได้หรือไม่?

ได้! แนบรูปถ่ายได้ถึง 8 ใบในแต่ละรายการ รูปถ่ายเก็บอย่างมีประสิทธิภาพพร้อมสร้าง thumbnail อัตโนมัติและซิงค์ข้ามอุปกรณ์ผ่าน iCloud

เทมเพลตบันทึกคืออะไร?

เทมเพลตคือ prompt สำเร็จรูปที่ช่วยเริ่มเขียนได้เร็ว สร้างเทมเพลตเองสำหรับรายการขอบคุณ การทบทวนรายวัน การติดตามเป้าหมาย หรือรูปแบบที่เกิดซ้ำ

ได้ ส่งออกรายการเป็น PDF, Excel หรือ JSON เพื่อสำรองข้อมูล แบ่งปันรายการที่เลือก หรือย้ายข้อมูลไปยังแพลตฟอร์มอื่น

ใช้งานบน Mac ได้หรือไม่?

ได้! แอปทำงานบน Apple Silicon Mac (M1 ขึ้นไป) รายการซิงค์อัตโนมัติระหว่าง iPhone, iPad และ Mac ผ่าน iCloud

เขียนหลายรายการต่อวันได้หรือไม่?

แน่นอน เขียนรายการได้มากเท่าที่ต้องการแต่ละวัน บางคนบันทึกตอนเช้าและเย็น ในขณะที่คนอื่นเพิ่มบันทึกตลอดวัน

มีโฆษณาหรือติดตามฉันหรือไม่?

ไม่มี เราลบโฆษณาทั้งหมดออกจากแอปแล้ว เราไม่ติดตามคุณในแอปหรือเว็บไซต์อื่น และไม่เคยขายข้อมูลของคุณ

ข้อมูลซิงค์ผ่าน iCloud เหมือน Apple Notes และ Photos รายการไดอารี่เป็นส่วนตัวทั้งในเวอร์ชันฟรีและพรีเมียม

ต้องสมัครสมาชิกหรือไม่?

แอปดาวน์โหลดและใช้งานได้ฟรีทั้งหมด ฟีเจอร์หลักทำงานได้โดยไม่ต้องสมัครสมาชิก การสมัครสมาชิกพรีเมียมปลดล็อกฟีเจอร์ AI ขั้นสูง การส่งออกไม่จำกัด และการปรับแต่งเพิ่มเติม
