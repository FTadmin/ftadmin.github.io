Γραφή Χωρίς Αποσπάσεις

Καθαρή, απλή διεπαφή που βάζει τις λέξεις σας πρώτες. Χωρίς ακαταστασία, μόνο εσείς και οι σκέψεις σας. Γράψτε όσο πολύ ή λίγο θέλετε κάθε μέρα.

Συναισθηματική Ανάλυση ΤΝ

Η αυτόματη ανίχνευση συναισθήματος δείχνει τον συναισθηματικό τόνο των καταχωρήσεών σας χρησιμοποιώντας το Natural Language API της Apple στη συσκευή. Πλήρως ιδιωτικό - δεν εγκαταλείπει ποτέ τη συσκευή σας.

Σάρωση ΤΝ

Σαρώστε αντικείμενα, έγγραφα, αποδείξεις ή κείμενο με την κάμερά σας και αφήστε την ΤΝ να τα περιγράψει και να εξαγάγει κείμενο. Δημιουργήστε αμέσως καταχωρήσεις ημερολογίου από πραγματικά αντικείμενα.

Ανάλυση Συχνότητας Λέξεων

Ανακαλύψτε τι σκέφτεστε πραγματικά. Δείτε ποιες λέξεις και θέματα εμφανίζονται συχνότερα στη γραφή σας για να εντοπίσετε μοτίβα και επαναλαμβανόμενες σκέψεις.

Ημερολόγια Φωτογραφιών

Επισυνάψτε έως 8 φωτογραφίες ανά καταχώρηση για να αποτυπώσετε αναμνήσεις οπτικά. Ιδανικό για ημερολόγια ταξιδιών, ημερολόγια ευγνωμοσύνης ή απλά για να τεκμηριώσετε την ημέρα σας.

Πρότυπα Σημειώσεων

Ξεκινήστε γρήγορα τις καταχωρήσεις σας με προσαρμόσιμα πρότυπα. Ιδανικό για ημερολόγια ευγνωμοσύνης, πρωινές σελίδες, παρακολούθηση στόχων ή οποιαδήποτε επαναλαμβανόμενη πρακτική καταγραφής ημερολογίου.

Σάρωση Αντικειμένων

Στρέψτε την κάμερά σας σε οποιοδήποτε αντικείμενο και αφήστε την ΤΝ να το περιγράψει με δημιουργικό πλαίσιο για το ημερολόγιό σας. Μετατρέψτε καθημερινές στιγμές σε ουσιαστικές καταχωρήσεις.

Σάρωση Κειμένου

Καταγράψτε κείμενο από έγγραφα, πινακίδες ή οθόνες με ΤΝ-ενισχυμένο OCR. Ψηφιοποιήστε αμέσως γραπτό περιεχόμενο στις καταχωρήσεις ημερολογίου σας.

Ισχυρή Αναζήτηση

Βρείτε οποιαδήποτε καταχώρηση αμέσως με αναζήτηση πλήρους κειμένου. Τα ημερολόγιά σας ευρετηριάζονται επίσης στο iOS Spotlight για γρήγορη πρόσβαση από οπουδήποτε στη συσκευή σας.

Πώς Λειτουργεί το Ημερολόγιο

Ο προσωπικός σας σύντροφος ημερολογίου για την καταγραφή στιγμών της ζωής και την ανακάλυψη μοτίβων

1. Γράψτε τις Σκέψεις σας

Άμεση Καταχώρηση: Ανοίξτε την εφαρμογή και αρχίστε να γράφετε αμέσως. Χωρίς ρύθμιση, χωρίς διαμόρφωση, χωρίς πολυπλοκότητα. Μόνο οι σκέψεις σας και η σελίδα. Η καθαρή, χωρίς αποσπάσεις διεπαφή σας βοηθά να επικεντρωθείτε στην έκφρασή σας.

Πλούσια Μέσα: Εμπλουτίστε τις καταχωρήσεις με φωτογραφίες, επισυνάψτε εικόνες για να αποτυπώσετε αναμνήσεις οπτικά. Προσθέστε ηχογραφήσεις, σκίτσα ή οποιαδήποτε μέσα που βοηθούν να αφηγηθείτε την ιστορία σας. Κάθε καταχώρηση γίνεται ένα πλήρες στιγμιότυπο της ημέρας σας.

Πρότυπα Σημειώσεων: Χρησιμοποιήστε έτοιμα πρότυπα για καθημερινές αναστοχάσεις, ημερολόγιο ευγνωμοσύνης, παρακολούθηση στόχων, αρχεία ονείρων ή δημιουργήστε τα δικά σας προσαρμοσμένα πρότυπα. Τα πρότυπα παρέχουν δομή ενώ σας δίνουν ακόμη την ελευθερία να γράφετε φυσικά.

2. Δείτε τα Μοτίβα να Αναδύονται

Συναισθηματική Ανάλυση: Η ΤΝ ανιχνεύει αυτόματα τον συναισθηματικό τόνο των καταχωρήσεών σας - θετικό, αρνητικό ή ουδέτερο. Δείτε τις τάσεις της διάθεσής σας απεικονισμένες σε εβδομάδες και μήνες για να κατανοήσετε τα συναισθηματικά σας μοτίβα.

Συχνότητα Λέξεων: Ανακαλύψτε για ποια θέματα, πρόσωπα ή θεματικές γράφετε περισσότερο. Η εφαρμογή εντοπίζει επαναλαμβανόμενες λέξεις και φράσεις, αποκαλύπτοντας τι κυριαρχεί στο μυαλό σας και πώς αλλάζει η εστίασή σας με την πάροδο του χρόνου.

Προβολή Ημερολογίου: Περιηγηθείτε σε ολόκληρο το ιστορικό καταγραφής σας σε μια διεπαφή ημερολογίου. Κάθε μέρα εμφανίζει ένα απόσπασμα και δείκτη διάθεσης. Μεταβείτε γρήγορα σε οποιαδήποτε παλαιότερη καταχώρηση για να δείτε τι σκεφτόσασταν εκείνη την ημέρα.

3. Αναστοχαστείτε και Αναπτυχθείτε

Πληροφορίες ΤΝ: Κάντε ερωτήσεις για το ημερολόγιό σας: «Ποιες ήταν οι πιο ευτυχισμένες μέρες μου τον περασμένο μήνα;» ή «Δείξε μου όλες τις καταχωρήσεις που ανέφερα τους στόχους μου.» Λάβετε άμεσες, περιεχόμενες απαντήσεις που σας βοηθούν να κατανοήσετε το ταξίδι σας.

Παρακολούθηση Προσωπικής Ανάπτυξης: Ελέγξτε πώς εξελίσσονται οι σκέψεις, οι ανησυχίες και οι φιλοδοξίες σας. Δείτε πρόοδο προς στόχους που αναφέρθηκαν σε παλαιότερες καταχωρήσεις. Εντοπίστε τομείς ανάπτυξης και γιορτάστε πόσο έχετε προχωρήσει.

Αναζητήστε Τα Πάντα: Η ισχυρή αναζήτηση βρίσκει οποιαδήποτε λέξη, φράση ή έννοια σε χρόνια καταγραφής ημερολογίου. Φιλτράρετε κατά ημερομηνία, διάθεση ή λέξεις-κλειδιά. Ολόκληρο το γραπτό ιστορικό σας είναι αμέσως προσβάσιμο και αναζητήσιμο.

4. Κρατήστε τα Πάντα Ασφαλή

Ιδιωτικό & Ασφαλές: Όλες οι καταχωρήσεις κρυπτογραφούνται και αποθηκεύονται στον προσωπικό σας λογαριασμό iCloud. Μόνο εσείς μπορείτε να διαβάσετε το ημερολόγιό σας - η εφαρμογή χρησιμοποιεί κρυπτογράφηση από άκρο σε άκρο ώστε τα λόγια σας να παραμένουν εντελώς ιδιωτικά.

Αυτόματο Αντίγραφο Ασφαλείας: Εβδομαδιαία αυτόματα αντίγραφα ασφαλείας JSON διασφαλίζουν ότι δεν θα χάσετε ποτέ τις σκέψεις σας. Χειροκίνητο αντίγραφο ασφαλείας ανά πάσα στιγμή για εξαγωγή ολόκληρου του ημερολογίου σας. Τα λόγια σας προστατεύονται από απώλεια ή βλάβη συσκευής.

Συγχρονισμός Μεταξύ Συσκευών: Γράψτε στο iPhone, συνεχίστε στο iPad, ελέγξτε στο Mac. Όλες οι καταχωρήσεις συγχρονίζονται αυτόματα μέσω iCloud. Το ημερολόγιό σας σας ακολουθεί παντού, πάντα ενημερωμένο σε κάθε συσκευή.

Αποποίηση ευθύνης ΑΙ

Οι γνώσεις, αναφορές και συστάσεις που δημιουργούνται από την ΤΝ προορίζονται μόνο για γενική πληροφόρηση και δεν αντικαθιστούν την ιατρική συμβουλή, διάγνωση ή θεραπεία. Τα πρότυπα που εντοπίζει η ΤΝ δείχνουν συσχετίσεις, αλλά η συσχέτιση δεν συνεπάγεται αιτιότητα. Να συμβουλεύεστε πάντα έναν γιατρό ή εξειδικευμένο πάροχο υγειονομικής περίθαλψης για οποιαδήποτε ιατρική ερώτηση ή ανησυχία. Αυτά τα εργαλεία έχουν σχεδιαστεί για να σας βοηθούν να οργανώνετε και να κατανοείτε τα δεδομένα υγείας σας, όχι για να παρέχουν ιατρική καθοδήγηση.

Συχνές ερωτήσεις

Είναι ιδιωτικό το ημερολόγιό μου;

Απολύτως. Οι καταχωρήσεις σας αποθηκεύονται κρυπτογραφημένες στη συσκευή σας και στον ιδιωτικό σας λογαριασμό iCloud. Δεν βλέπουμε ποτέ τη γραφή σας. Ακόμα και η συναισθηματική ανάλυση ΤΝ γίνεται εντελώς στη συσκευή σας χρησιμοποιώντας το Natural Language API της Apple - τα λόγια σας δεν εγκαταλείπουν ποτέ το τηλέφωνό σας.

Πώς λειτουργεί η συναισθηματική ανάλυση;

Όταν γράψετε 8 ή περισσότερες λέξεις, η εφαρμογή χρησιμοποιεί το ενσωματωμένο Natural Language framework της Apple για να αναλύσει τον συναισθηματικό τόνο. Εμφανίζεται ένα emoji για να δείξει αν η καταχώρηση διαβάζεται ως θετική, αρνητική ή ουδέτερη. Αυτό γίνεται εντελώς στη συσκευή για πλήρη ιδιωτικότητα.

Τι είναι η ανάλυση συχνότητας λέξεων;

Η λειτουργία Τάσεις αναλύει όλες τις καταχωρήσεις σας για να δείξει για ποιες λέξεις και θέματα γράφετε πιο συχνά. Αυτό σας βοηθά να εντοπίσετε επαναλαμβανόμενα θέματα, ανησυχίες ή ενδιαφέροντα στη σκέψη σας.

Μπορώ να προσθέσω φωτογραφίες στο ημερολόγιό μου;

Ναι! Μπορείτε να επισυνάψετε έως 8 φωτογραφίες σε κάθε καταχώρηση ημερολογίου. Οι φωτογραφίες αποθηκεύονται αποτελεσματικά με αυτόματη δημιουργία μικρογραφιών και συγχρονίζονται σε όλες τις συσκευές σας μέσω iCloud.

Τι είναι τα πρότυπα σημειώσεων;

Τα πρότυπα είναι προγραμμένες προτροπές που σας βοηθούν να ξεκινήσετε να γράφετε γρήγορα. Δημιουργήστε προσαρμοσμένα πρότυπα για πράγματα όπως λίστες ευγνωμοσύνης, καθημερινές αναθεωρήσεις, παρακολούθηση στόχων ή οποιαδήποτε άλλη επαναλαμβανόμενη μορφή χρησιμοποιείτε.
